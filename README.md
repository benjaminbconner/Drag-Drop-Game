Drag Drop Puzzle Game
A small, accessible, touch‑friendly JavaScript puzzle where players drag numbered pieces into matching slots. Built with plain HTML, CSS, and JavaScript, the game includes pointer/touch support, keyboard controls, scoring, and progressive levels so you can extend or embed it quickly.

Features
Touch and pointer support using Pointer Events for consistent mobile and desktop behavior.

Keyboard accessibility: pick up and drop pieces with Enter/Space and navigate with arrow keys.

Scoring and levels: points for correct placements, penalties for mistakes, and level progression.

Simple, dependency‑free codebase: no libraries required — easy to read and modify.

ARIA attributes and live region to communicate state to assistive technologies.
